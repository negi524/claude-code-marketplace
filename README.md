# Claude Code Marketplace

個人的なClaude Codeマーケットプレイス - スキルのコレクション

## インストール

```bash
# ローカルディレクトリから
/plugin marketplace add /path/to/claude-code-marketplace

# GitHubリポジトリから（公開している場合）
/plugin marketplace add owner/repo
```

## ディレクトリ構造

```
claude-code-marketplace/
├── .claude-plugin/
│   └── marketplace.json
└── skills/
```
